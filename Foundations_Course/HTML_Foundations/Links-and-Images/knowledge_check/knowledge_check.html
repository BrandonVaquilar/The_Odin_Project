Knowledge Check

    What element is used to create a link?

  • Answer: The anchor element <a>

    • What is an attribute?

  • Answer: An attribute gives more information to an element, and always goes in the element's opening tag.

    • What attribute tells links where to go to?

  • Answer: the href attribute

    • What security considerations must be taken if you wish to use the target attribute to open links in a new tab/window?

  • Answer: the rel attribute value noopener, so the newly opened page cannot interact with the original page, and the value norefferer which hides the user or refferer from the new page. norefferer also has the behavior of noopener.

    • What is the difference between an absolute and relative link?

  • Answer: An absolute link is the absolute path from where it is found on the internet. A relative link is relative to the local files from where it is found.

    • Which element is used to display an image?

  • Answer: <img>

    • What two attributes do images always need to have?

  • Answer: <img> is requires for an image, and <alt> is good to always have for extra information for browsers.

    • How do you access a parent directory in a filepath?

  • Answer: You can access the current directory with . and you can access the parent directory with ..

    • What are the four main image formats that you can use for images on the web?

  • Answer: jpg, png, gif, svg
